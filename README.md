# Contact Book v1.0 (C)

## Description
A small console app written in C that allows users to store and persist contacts using a CSV file.
This is my first programming (mini)project, built as part of my learning process while following CS50.

## Features
- Add contacts
- List contacts
- Save to CSV
- Load from CSV
